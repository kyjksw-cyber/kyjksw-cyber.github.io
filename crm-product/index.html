FAQ

자주 묻는 질문

기존 ERP와 연동이 가능한가요? ∨
네, SAP, 더존 iCUBE, 영림원 등 국내외 주요 ERP 14종과 커넥터를 기본 제공합니다. 온프레미스 환경도 VPN 터널 방식으로 연동 가능하며, 커스텀 ERP는 API 명세서 제공 시 별도 개발로 지원합니다. Enterprise 플랜은 전담 엔지니어가 직접 연동 작업을 진행합니다.
도입 기간은 얼마나 걸리나요? ∨
평균 도입 기간은 14일입니다. 1~3일: 요구사항 파악 및 계정 세팅, 4~10일: ERP·채널 연동 및 데이터 검증, 11~14일: 온보딩 교육 및 운영 전환. ERP 연동 없이 기본 대시보드만 사용하는 경우 3~5일 내 시작 가능합니다.
데이터 보안은 어떻게 관리되나요? ∨
Pulse는 ISO 27001 정보보안 인증을 취득했습니다. 모든 데이터는 AES-256 암호화로 저장되며, 전송 구간은 TLS 1.3을 적용합니다. 고객 데이터는 한국 데이터센터(KT Cloud)에만 저장되며, 제3자 공유는 일절 없습니다. Enterprise 고객은 전용 격리 환경(Dedicated Tenant)으로 운영됩니다.
카카오 알림톡 외에 어떤 채널을 지원하나요? ∨
현재 지원 채널: 카카오 알림톡, 카카오 친구톡, LMS/SMS, 앱푸시(iOS/Android), 이메일, 웹푸시. Pro 이상 플랜에서 멀티채널 통합 관리가 가능하며, 한 고객에게 여러 채널로 발송한 경우에도 중복 전환을 자동으로 제거해 정확한 기여도를 산출합니다.
무료 체험 기간 이후 자동 결제되나요? ∨
1개월 무료 체험 기간 종료 후 자동 결제가 진행되지 않습니다. 체험 종료 3일 전 이메일로 안내드리며, 별도 의사 표시가 없으면 플랜이 자동으로 만료됩니다. 플랜 유지를 원하시면 결제 정보를 등록하시면 됩니다. 계약 후에도 위약금 없이 30일 전 해지 통보로 언제든 해지 가능합니다.
기술팀 없이도 운영할 수 있나요? ∨
네, Pulse는 마케팅 담당자가 직접 운영할 수 있도록 설계됐습니다. ERP 초기 연동은 저희 팀이 전담 처리하며, 이후 일상적인 캠페인 운영은 코드 없이 UI만으로 가능합니다. 모든 플랜에 온보딩 영상 및 사용 가이드가 포함되고, Pro 이상은 채팅 지원도 제공됩니다.
